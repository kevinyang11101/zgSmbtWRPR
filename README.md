## 核心功能说明

系统通过以下核心功能，实现对宠物医院业务的有效管理：

- **预约管理：** 支持用户在线预约，医生端管理预约记录。
- **宠物管理：** 用户可记录宠物日志，便于追踪宠物健康状况。
- **用户与角色管理：** 系统管理员可管理用户与医生账户，配置角色权限。

通过这些功能的组合，本系统旨在提高宠物医院的运营效率和服务质量。
